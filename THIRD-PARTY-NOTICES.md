# Third-party notices

## MAC-vendor (OUI) database — `data/oui.tsv`

The bundled vendor lookup table is derived from the Wireshark project's `manuf`
database (the IEEE OUI / MA-L / MA-M / MA-S registry as curated by Wireshark).

- Source: the Wireshark `manuf` data.
- Licence: GPLv2 (as distributed by Wireshark).

Only the address-prefix → vendor-name mapping is reproduced, reformatted as a
tab-separated file. No Wireshark code is included.
